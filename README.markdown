# Biljkich + Arduino

## What it is
Internet-of-things connected *Spathiphyllum sp.* houseplant. Uses the latest cosm.com library.
Currently has two sensors:
 * Photoresistor for ambient light
 * Two zinc-plated nails for soil moisture
